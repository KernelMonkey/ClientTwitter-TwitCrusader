##ArchLinux - Install or Compile Package

### PKGBUILD
    $ su
    # makepkg --asroot
    # pacman -U twitcrusader-git-XXXXXXXXXXXXXX-1-i686.pkg.tar.xz

### AUR
    $ su
    # yaourt -S twitcrusader-git
