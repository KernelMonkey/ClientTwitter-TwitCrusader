#TwitCrusader++
###TwitCrusader++ is Twitter Client For Linux Desktop

##Required

* g++ >= 4.6.1
* gtkmm-3.0 >= 3.0.1
* oauth >= 0.9.4
* libxml-2.0 >= 2.6.32
* libnotify >= 0.7.3

## How to Install - Archlinux
    
### Dependencies:
    $ su
    # pacman -Syu
    # pacman -S gtkmm3 liboauth libxml2 libnotify


### Compile - Makefile
    $ ./autogen.sh
    $ ./configure
    $ make
    $ su
    # make install

#### Unistalling - Makefile
    $ su
    # make uninstall
